# Diary Admin

Админка для преподавателей

В рамках проекта хотим реализовать функционал:

- Создание групп
- Создание преподов
- Создание предметов
- Авто перенос расписания на неделю и тд
- ДЗ
- Оценки за урок
- Система подгрупп
